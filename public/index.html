About the Project

This MERN Fitness Tracker allows users to log workouts, browse exercises by body part, and watch YouTube tutorials.

  • Secure authentication with JWT
  • Workout logging with sets, reps, and notes
  • API-powered exercise library
  • Responsive design for desktop, tablet, and mobile
